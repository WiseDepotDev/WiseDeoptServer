# 慧仓智控（WiseDeoptServer）

## 项目简介

“慧仓智控”是一个用于智能仓储场景的智能调度与控制服务端项目。本项目主要作为竞赛与展示使用，聚焦于仓储优化、任务调度等算法与服务端实现。
